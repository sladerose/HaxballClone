# NeonBall - Haxball Clone

A multiplayer browser-based physics game inspired by Haxball, featuring a sleek "Neon" aesthetic.

## Features

- **Physics-driven Gameplay**: Realistic ball and player collisions using a custom physics engine.
- **Multiplayer Ready**: Integrated network logic for real-time play.
- **Modern UI**: Styled with CSS and the "Outfit" font from Google Fonts.
- **Customizable**: Easy to tweak physics constants and visual styles.

## Project Structure

- `index.html`: Main game entry point.
- `game.js`: Core game loop and state management.
- `physics.js`: Collision detection and movement logic.
- `network.js`: Multiplayer synchronization.
- `bot.js`: Basic AI implementation.
- `style.css`: Visual styling and layout.

## How to Play

1. Open `index.html` in your web browser.
2. Control your player with the WASD keys (or arrow keys, depending on current configuration).
3. Use the spacebar to kick the ball.
4. Goal: Kick the ball into the opponent's net to score!
